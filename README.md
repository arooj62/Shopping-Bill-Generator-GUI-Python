# Shopping-Bill-Generator-GUI-Python
## Description
A beginner-friendly Python project that calculates shopping bills and applies discounts based on the total purchase amount.
Shopping Bill Generator is a beginner-friendly desktop application developed using Python and Tkinter. The software enables users to select shopping items, enter quantities, manage a shopping cart, calculate taxes automatically, and generate a final bill through a simple graphical user interface.

This project was developed as part of my internship at **M Tech Production & Marketing** to practice Python programming and GUI development.

## Features
- Graphical User Interface (Tkinter)
- Customer-friendly shopping interface
- Item selection
- Quantity input
- Shopping cart
- Automatic total calculation
- 5% tax calculation
- Bill number generation
- Receipt display
- Input validation
- Error handling

## Tax Rule

The application automatically adds **5% tax** to the subtotal before displaying the final payable amount.
